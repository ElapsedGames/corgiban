Steps:
1. Implement level parsing/serialization in packages/core/src/encoding; enforce MAX_GRID_WIDTH/MAX_GRID_HEIGHT/MAX_BOXES via packages/shared/src/constraints.ts in parseLevel.
   - parseLevel(def: LevelDefinition): LevelRuntime (LevelDefinition comes from @corgiban/levels).
   - Token mapping table:
     Token | floor | occupant | notes
     W | - | - | wall
     E | floor | empty |
      | floor | empty | internal whitespace only
     T | target | empty |
     P | floor | player |
     B | floor | box |
     S | target | box | decompose; not a new cell kind
     Q | target | player | serialize player-on-target as Q
   - Trim leading/trailing whitespace from each row before tokenizing; internal spaces count as floor+empty occupant cells.
   - Enforce exactly one player, rectangular rows, and unknown-token rejection with descriptive errors.
   - serializeLevel(level: LevelRuntime): LevelDefinition['rows'] must invert parseLevel for the initial layout:
     - Every non-wall cell serializes to its canonical token (W, E, T, P, B, S, Q).
     - Q is emitted when a player occupies a target cell.
     - parseLevel({ id, name, rows: serializeLevel(level) }) must reproduce an equivalent LevelRuntime.
2. Add hashing utilities in packages/core/src/hashing:
   - normalize(state, width): string must return a deterministic string key, for example: "p:|b:".
   - The key must be injective: distinct (playerIndex, sorted boxIndices) pairs must always produce distinct strings.
   - Box indices are computed as row * gridWidth + col and sorted ascending.
   - Include player position and sorted box positions only.
   - Exclude history, move counts, and other metadata from the canonical key.
   - Implement hash(key: string): number as a deterministic 32-bit unsigned hash derived from normalize().
   - Never rely on hash alone for solver correctness; always compare the canonical key for equality.
3. Add unit tests for parsing constraints, serialization round-trip, and hashing normalization/hashing.
4. Export the public API from packages/core/src/index.ts:
   - Types: Cell, FloorType, Occupant, Direction, Position, LevelRuntime, GameState
